# Projet Management App

Application de gestion de projets et tâches avec authentification.

## Technologies
- **Frontend** : React, TailwindCSS
- **Backend** : Node.js, Express
- **Base de données** : MongoDB, Mongoose
- **Auth** : JWT
- **Email** : Nodemailer

## Installation

### Backend
```bash
cd server
npm install
cp .env.example .env   # et remplir les variables
npm start
